# Generate NFT collection

## Installation

- Rename `.env.sample` to `.env` and adjust environment variables.

```sh
pnpm install
```

## Usage

- Adjust system prompt and first instruction in `scripts/createCollection.ts`.

```sh
pnpm execute
```

NFTs will be generated in `./projects/nfts`.
